# Flutter CRUD Product

Aplikasi sederhana menggunakan Flutter untuk mengelola data produk.
Pengguna dapat menambahkan, mengedit, dan menghapus data produk secara langsung melalui tampilan aplikasi.

## Fitur

* Menampilkan daftar produk
* Menambahkan produk
* Mengedit produk
* Menghapus produk
* Menampilkan gambar dari URL

## Penjelasan

Data produk disimpan sementara di dalam aplikasi (belum menggunakan database), sehingga data akan kembali ke kondisi awal saat aplikasi ditutup.

Setiap produk memiliki atribut berupa nama, harga, dan gambar.
Saat menambah atau mengedit produk, akan muncul form dalam bentuk dialog untuk memasukkan data. Gambar yang dimasukkan melalui URL akan langsung ditampilkan sebagai preview.

## Catatan

Aplikasi ini dibuat untuk latihan dasar Flutter dalam memahami konsep CRUD dan penggunaan StatefulWidget.
